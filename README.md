# Vue 3 + Vite Template

Vue 3 starter project using Vite and Tailwind CSS.

## Tech Stack
- Vue 3
- Vite
- Tailwind CSS
- Vue Router
- ESLint
- Prettier

## Setup

```bash
npm install
npm run dev
npm run build
npm run preview

src/
 ├── assets/
 │   └── style.css
